## 七、注意事项

1. **文件名不要用中文** — GitHub Pages（Jekyll）处理中文文件名会 404
2. **Markdown 文件需要 front-matter** — 文件开头必须有 `---\nlayout: default\n---` 才能被 Jekyll 渲染成网页
3. **每次推送后等 1-2 分钟** — Jekyll 构建需要时间，刚推完访问可能还是旧版
4. **index.html 不走 Jekyll** — 首页是纯 HTML，修改后直接生效
5. **图片路径** — 教程里引用图片用 `images/xxx.png`，模板图片用 `tpl-` 前缀
